# UT3-A2 Elige tu IDE ideal


Actividad diseñada para escenario telemático


## Objetivos
- Analizar y comparar distintos entornos de desarrollo.  
- Reflexionar sobre los criterios que influyen al elegir un IDE.  
- Justificar una elección técnica de forma razonada.






## Contexto
Imagina que formas parte de un nuevo equipo de desarrollo que va a iniciar un proyecto web o de aplicaciones.  
Antes de empezar, el equipo debe decidir **qué IDE utilizar**.  
Tú eres la persona encargada de **analizar varias opciones y justificar la elección final** ante el grupo.






## Tareas


### Investigación rápida (15 min)
Busca información sobre **tres IDE distintos** (pueden ser de lenguajes o entornos diferentes).  
Algunos ejemplos:
- Visual Studio Code  <=
- IntelliJ IDEA  
- Eclipse  <=
- NetBeans  <=
- PyCharm  
- Android Studio  
- Replit o Gitpod (IDE online)






### Análisis comparativo (20 min)
Completa una tabla con la siguiente estructura:


| Criterio                                                                         | Visual Studio Code                                                                                     | Eclipse                                                                                                        | NetBeans                                                                                                       |
|:---------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------|
| Lenguajes principales que soporta                                                | JavaScript, TypeScript, Python, Java, C/C++, PHP, Go, Rust, etc.                                       | Java, C/C++, Python, PHP, JavaScript                                                                           | Java, PHP, C/C++, HTML5, JavaScript                                                                            |
| Peso o consumo de recursos                                                       | Ligero y rápido; depende de las extensiones instaladas                                                | Pesado; alto consumo de memoria y CPU                                                                          | Intermedio; más ligero que Eclipse, pero más pesado que VS Code                                              |
| Facilidad de instalación                                                        | Muy fácil; instalador rápido y multiplataforma                                                       | Requiere configuración inicial; instalación más compleja                                                    | Instalación sencilla, incluye JDK en algunos paquetes                                                         |
| Funciones destacadas (depuración, refactorización, integración con Git, etc.) | Depuración integrada, IntelliSense, integración con Git, extensiones, terminal integrada, Live Share | Depuración avanzada, refactorización, integración con Maven y Git, herramientas para desarrollo empresarial | Buen soporte para depuración, refactorización, integración con Maven y Git, herramientas gráficas para GUI |
| Interfaz / usabilidad                                                            | Moderna, personalizable y minimalista                                                                  | Más tradicional y técnica, menos intuitiva                                                                   | Interfaz clara, aunque algo anticuada                                                                          |
| Comunidad y soporte                                                              | Amplia comunidad, actualizaciones frecuentes de Microsoft                                              | Gran comunidad, abundante documentación                                                                       | Comunidad activa, soporte de Apache y documentación oficial                                                   |
| Licencia / coste                                                                 | Gratuito y de código abierto (MIT License)                                                            | Gratuito y de código abierto (EPL License)                                                                    | Gratuito y de código abierto (Apache License 2.0)                                                             |




### Conclusión y justificación (10 min)
Redacta un texto breve (unas 10 líneas) donde respondas:


- ¿Cuál de los tres IDE recomendarías para tu grupo de trabajo?
- ¿Por qué crees que se adapta mejor a un proyecto de desarrollo web o de aplicaciones?  
- ¿Qué criterio has valorado más (rendimiento, comodidad, lenguaje, integración, etc.)?


Recomendaría Visual Studio Code para mi grupo de trabajo, ya que es un entorno ligero, rápido y muy flexible. Su soporte para múltiples lenguajes y la gran cantidad de extensiones disponibles lo hacen ideal para proyectos de desarrollo web y de aplicaciones modernas. Además, ofrece una excelente integración con Git, depuración integrada y herramientas colaborativas como Live Share, que facilitan el trabajo en equipo. En comparación con Eclipse y NetBeans, su interfaz es más intuitiva y personalizable, lo que mejora la productividad. Por estas razones, VS Code se adapta mejor a entornos dinámicos donde se requiere agilidad, colaboración y soporte para diversos lenguajes de programación.
